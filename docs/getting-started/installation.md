# Installation

## Prerequisites

- Node.js >= 18
- npm

## From npm (recommended)

```bash
npm install -g @mobrienv/autoloop
```

## From source

```bash
git clone https://github.com/mikeyobrien/autoloop.git && cd autoloop
npm install
npm run build
node bin/autoloop --help
```

## Verify installation

```bash
autoloop --help
```
